Optimize Image Files from the Terminal

29 Jun 2016

At several times, you’ll find yourself in need for rapid optimization for a set of images at your hand. There are nice handy utility libraries that can help reduce image sizes with a single command in your terminal. The following lists some of them:

  • OptiPNG: Advanced PNG Optimizer: Get it form Mac OS X:
brew install optipng

for Linux Ubuntu: sh apt-get install optipng

To optimize an image, run:

optipng <img-name.png>

or wildcard all recursively of in current directory and sub directories:

find . -iname "*.png" -exec optipng -o7 {} \;
  • jpegoptim:
brew install jpegoptim

for Linux Ubuntu: sh sudo apt-get install jpegoptim

To optimize an image, run the the next command with -m flag set the maximum image quality (0-100):

jpegoptim -m70 <img-name.png>

or optimize all images in the current and sub directories:

find . -iname "*.jpg" -exec jpegoptim -m80 -o -p {} \;
  • gifsicle:
brew install gifsicle

for Linux Ubuntu: sh sudo apt-get install gifsicle

The commands for single image, and a group of images respectively, are: ```sh # for single image. gifsicle –batch -V -O2 .gif

to start from cureent dir and go down.

find . -iname “*.gif” -exec gifsicle –batch -V -O2 {} \; ```

Aliases

Here are some aliases to keep commands nearby for fast run on the active directory images.

Be warn that those as with the mentioned previous commands are all alter the original images. sh alias compress_png="optipng -o7 *.png" alias compress_jpg="jpegoptim -m80 *.jpg" alias compress_gif="gifsicle --batch -V -O2 *.gif" Note that you can tweak the optimization level and other options to suit your needs.

ImageOptim

. If you like to checkout a GUI app, ImageOptim is a pretty open source one for Mac OS X that supports multiple image formats.

Happy Optimization :) !
